About

More About Me.

An autodidactic, I discovered Cocoa and Objective-C almost a decade ago through work on an open-source project. With the emergence of iOS, I’ve been lucky enough to earn a living using this skill.

Working on the Apple Platform for so long makes for a deep understanding of framework conventions, best practices, memory management, and development tools. When it comes to code and application architecture, I’m passionate about quality and maintainability.

Along with working full-time at a class-leading company, I work independently on select contract projects. I’m as much a designer as a developer, crafting every UI pixel in my work.
